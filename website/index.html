Installation Guide

Docker Deployment 
# Clone the repository
git clone https://github.com/obi-wan-jr/devopsAi.git
cd devopsAi

# Deploy to Raspberry Pi 5
./scripts/docker-deploy.sh

# Or manually with Docker Compose
docker-compose -f docker-compose.pi.yml up -d

Docker deployment is the recommended method for easy setup and management.

Manual Installation 
# Update system packages
sudo apt update && sudo apt upgrade -y

# Install dependencies
sudo apt install -y build-essential cmake git python3 python3-pip python3-venv

# Clone and setup
git clone https://github.com/obi-wan-jr/devopsAi.git
cd devopsAi
chmod +x scripts/install.sh
./scripts/install.sh

# Start the service
sudo systemctl start ai-sysadmin-agent

Troubleshooting

Service Won't Start

Check logs:

sudo journalctl -u ai-sysadmin-agent -f

Verify Python environment:

cd /home/inggo/ai-agent
source ai-agent-env/bin/activate
python --version
pip list | grep autogen

Model Loading Errors

Check model file:

ls -la /home/inggo/ai-agent/models/
file /home/inggo/ai-agent/models/qwen2-1.5b-q4_k_m.gguf

Re-download model:

./scripts/setup_model.sh

Web Interface Not Accessible

Check if service is listening:

sudo netstat -tlnp | grep :8080

Check firewall:

sudo ufw status
sudo ufw allow 8080/tcp
